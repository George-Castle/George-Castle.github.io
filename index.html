Advanced Computer Graphics

In the second term of my MSc I undertook an Advanced Computer Graphics module. The module required the completion of two courseworks: the first of which assessed our ability to produce a simple procedural image-based lighting model using surface normal and reflection mappings, creating a resterised latlong spherical map for imaged-based lighting, shwon above. The second coursework required the creation of Fresnel Reflectance plots and Monte Carlo samples according to the illumination generated from an environment map of the Grace Cathedral.

Python Quantum Computing

Using the ProjectQ open source Quantum Computer I decided to try and learn more about quantum computing by implementing some simple programs. First I created a random coin toss game, that used Qubits in a superposition to create the random result of heads or tails and measured the probability of which result the quantum computer would create. After this I simulated sending a message between two recipients using two entangled Qubits in a Bell-pair. I succeded sending and decoding a message, I then went on the test the encryption of this message if it were intercepted by a third party trying to read the original message without access to the entangled Qubits. The output is displayed above, showing how message sending using entangled Qubits has in-built end to end encryption and cannot be intercepted without an entangled Qubit.

Refining Gaze Prediction for Collaboration within a Virtual Environment

For the Final Project of my Advanced Computing MSc I contributed to the field of gaze prediction within virtual reality (VR). This involved analysing current bilogical gaze research and contemporary VR gaze research in attempt to reduce the possible cone of vision of a collaborator based on types of head movements. The main contributions of the paper consist of: a novel reduction to a users cone of vision when turning left or right and a machine learning algorithm that is able to predict a users gaze to a higher degree of accuracy than current gaze research; the algorithm was then implemented within a virtual testing environment built in Unity, which was then tested on six users. The repository for this project will remain unavailable until marking and moderation has been completed.

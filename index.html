OpenGL Space Scene

For the final coursework of my BSc Computer Graphics module I created an extra-terrestrial scene, using C++ and OpenGL; featuring significant events in space exploration. I used a variety of graphical techniques to create this scene such as surface subdivision and keyframe animation, which I combined with SUVAT equations to simulate the moons gravity for realistic movement. I received a final grade of 95% for this coursework. My code and report can be viewed using the link below.

ATAK Plugin Development

Spent the last two years developing plugins for the Android application ATAK (Android Team Awareness Kit); contracted by the MOD, whilst working at BAE systems. The plugins are developed using Kotlin, Java, Room, TensorFlow etc. to create a variety of user requested features, such as an optical character recognition machine learning model for vehicle registration recognition, a real-time vocal channel language translator and live satellite tracking for satllite dish placement. The project also requires a week of each month to be spent living and working on client site to ellicit feature requirements from end users and address any product or user issues. Several of the plugins (Voice, TAKSat, TDAL, PDF etc.) have been deployed to the Google Playstore for the use of the general public.

ISM in-Orbit Servicing and Satellite Upgradability

Completed a STRETCH assignment for subsidary company In-Space Missions, whilst at BAE, to design and prototype a satellite servicing vehicle to autonomously repair and upgrade satellites in low earth orbit. As the software lead on the robotics team, I developed a computer vision model, using Python and TensorFlow Lite, for satellite naivigation throughout the soft and hard docking process. TensorFlow Lite enabled the model to be lightweight and run with minimal processing power whilst boasting 84% accuracy on the training data.

Refining Gaze Prediction for Collaboration within a Virtual Environment

My Advanced Computing MSc research thesis contributed to the field of gaze prediction within virtual reality (VR). This involved analysing current bilogical gaze research and contemporary VR gaze research to reduce the possible cone of vision of virtual collaborators using head movement characterisations. The two primary contributions presented in the paper are: a novel reduction to a VR users cone of vision when turning left or right and a machine learning (ML) model, developed using Python and TensorFlow lite. The ML model is able to predict a users gaze to a higher degree of accuracy than current gaze research. The gaze prediciton model was then implemented within a virtual testing environment, built in Unity using C#, and was tested on six users. The full report can be viewed on my Github page linked below.
